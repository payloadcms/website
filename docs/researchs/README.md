## 핵심 인사이트 요약

### 2026년 정부지원사업 기회
- 전체 예산: 16.5~17조원
- **혁신 소상공인 AI활용 지원**: 114억원 (신규)
- **스마트상점 기술보급**: SaaS형 100% 정부지원
- **TIPS 개편**: R&D 최대 8억원, 선투자 2억원+

### 기술 전략 핵심
- Palantir Ontology 패턴 (Object-Link-Action) 적용
- VAN 결제 데이터 → Digital Twin 구축
- "매장 팔란티어" = 소상공인용 데이터 운영체제

### 경쟁 환경
- 마이프차: 프랜차이즈 정보 플랫폼 (데이터 역공학 대상)
- 토스: 결제 생태계 확장 → VAN사 위협
- 나이스정보통신: VAN 시장점유율 27.8%
