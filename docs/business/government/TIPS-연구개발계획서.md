### 온톨로지 스키마 (Neo4j)

```cypher
// 핵심 노드
```

# SLIDE 8: 기술 혁신성 (3/3)

## 혁신 3: 하이브리드 AI 아키텍처

### 비용 효율적 AI 구조

```python
class CareOnHybridAI:
    """
    로컬 AI (90%) + 클라우드 AI (10%) = 비용 90% 절감
    """

    def process(self, query):
        # Step 1: 도메인 지식 여부 판별
        if self.is_domain_knowledge(query):
            # 로컬 AI 처리 (빠름, 비용 0)
            return self.local_ai.respond(query)
        else:
            # 클라우드 AI 호출 (GPT/Claude)
            return self.cloud_api.respond(query)

    def is_domain_knowledge(self, query):
        """
        ✓ "칠성사이더 가격?" → 로컬 (제품 DB)
        ✓ "어제 매출?" → 로컬 (거래 DB)
        ✓ "인기 메뉴 3개?" → 로컬 (분석 결과)
        ✗ "2026년 세금 신고?" → 클라우드
        ✗ "마케팅 문구 작성해줘" → 클라우드
        """
        return self.classifier.predict(query) == "DOMAIN"
```

### 하이브리드 AI의 장점

| 구분 | 클라우드 Only | 하이브리드 |
|------|-------------|-----------|
| **API 비용** | 월 100만원+ | **월 10만원** |
| **응답 속도** | 1~3초 | **0.1초 (로컬)** |
| **오프라인** | 불가 | **가능 (로컬)** |
| **데이터 보안** | 외부 전송 | **민감정보 로컬** |
