# CareOn 2026년 복수 지원 가능 정부지원사업 리스트

> **작성일**: 2025-12-17
> **목적**: CareOn이 동시 또는 순차 지원 가능한 정부지원사업 종합 정리

---

## 핵심 요약

CareOn은 창업 3년 이내 AI 기반 소상공인 솔루션 스타트업으로, **최대 4~5개 사업 동시 지원**이 가능합니다.

| 구분 | 최대 확보 가능 자금 |
|------|-------------------|
| 창업지원사업 (초기창업패키지) | 1억원 |
| R&D 지원 (TIPS/디딤돌) | 2~8억원 |
| 스마트상점 기술공급기업 등록 | 매출 확대 (정부 마케팅) |
| 정책자금 대출 | 최대 10억원 (저금리 2~3%) |
| **총 예상 확보 가능액** | **11~19억원+** |

## 1순위: 즉시 준비/신청 필요 사업

### 1-1. 초기창업패키지 (최대 1억원)

| 항목 | 내용 |
|------|------|
| **예산** | 455억원 (490개사) |
| **지원금** | 최대 **1억원** (평균 0.7억원) |
| **지원 대상** | 창업 후 **3년 이내** 법인 |
| **예상 공고** | 2026년 1~2월 |
| **주관** | 창업진흥원 (K-Startup) |

**준비사항**:
- 사업계획서 (AI 기술 혁신성 강조)
- MVP/프로토타입 증빙
- 팀 구성 및 역량 자료

**CareOn 적합도**: ⭐⭐⭐⭐⭐ (핵심 타겟)

### 1-2. 스마트상점 기술공급기업 등록

| 항목 | 내용 |
|------|------|
| **예산** | 약 11,000개 상점 지원 |
| **지원 효과** | 정부가 마케팅 비용 **50~70% 대납** |
| **SaaS형** | **국비 100%** 지원 (소상공인 부담 0%) |
| **예상 공고** | 2025년 12월 말 ~ 2026년 1월 |
| **주관** | 소상공인시장진흥공단 |

**등록 요건**:
- 상용화된 스마트기술 공급 가능
- 가격 적정성 검증 통과
- 2년 무상유지보수 기준 충족
- 기술 관련 특허/인증 보유 권장

**CareOn 적합도**: ⭐⭐⭐⭐⭐ (POS+매출분석 = '경영(매출분석)' 분야 직접 해당)

**담당 사이트**: [소상공인스마트상점](https://www.sbiz.or.kr/smst) / 문의 1600-6185

## 복수 지원 전략 (동시 신청 가능 조합)

### 권장 조합 A: 창업+기술보급 (동시 가능)

```
초기창업패키지 (1억원)
    +
스마트상점 기술공급기업 등록 (매출 확대)
    +
혁신 소상공인 AI활용 지원 (고객사 연계)
```

**예상 효과**:
- 직접 자금: 1억원
- 정부 마케팅 지원으로 고객 확보 비용 50~70% 절감
- 고객사(소상공인)의 AI POS 도입 장벽 제거

---

### 권장 조합 B: 창업+R&D (순차 가능)

```
초기창업패키지 (1억원) [2026년 상반기]
    →
TIPS (R&D 8억원) [선투자 2억원 확보 후]
    또는
디딤돌 (R&D 2억원) [R&D 첫 수행 시]
```

**예상 효과**:
- 총 3~9억원 확보
- 기술 고도화 및 특허 확보

---

### 권장 조합 C: 전방위 확장 (고성장 시)

```
초기창업패키지 → TIPS → 유니콘브릿지
    +
스마트상점 기술공급기업 + 글로벌 소상공인 육성
    +
정책자금 대출 (최대 10억원)
```

**예상 효과**: 최대 **19억원+** 정부 지원 확보

## 복수 지원 주의사항

### 중복 지원 제한 확인 필요

| 사업 | 중복 제한 여부 |
|------|--------------|
| 초기창업패키지 | 예비창업패키지와 중복 불가 |
| TIPS | 창업패키지와 **동시 수행 가능** |
| 디딤돌 | TIPS와 중복 제한 (선택 필요) |
| 스마트상점 기술공급기업 | 창업사업과 **병행 가능** |

### 지원 일정 타임라인

```
2025년 12월
    └── 스마트상점 기술공급기업 모집 공고 확인
    └── TIPS 운영사 컨택 시작

2026년 1월
    └── 스마트상점 기술공급기업 신청
    └── 소상공인 지원사업 통합공고 확인

2026년 2월
    └── 초기창업패키지 공고 → 신청
    └── 예비창업패키지 공고

2026년 3월
    └── 초기창업패키지 접수 마감
    └── 디딤돌 R&D 공고

2026년 상반기
    └── 혁신 소상공인 AI활용 지원 공고 예상
```
